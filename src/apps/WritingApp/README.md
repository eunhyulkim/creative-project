# 테크니컬 라이터

**테크니컬 라이터+++** 는 테크니컬 라이팅을 돕는 에디터입니다.  
React로 개발하였으며 일반적인 에디터 UI에 린터 기능을 추가했습니다.  
실시간으로 에러 메시지를 받거나 전체 분석 결과에 따른 리포트를 확인할 수 있습니다.
